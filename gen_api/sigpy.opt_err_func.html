sigpy.opt_err_func¶

sigpy.opt_err_func(params, x, y, func)¶

Error function for fitting a function using non-linear optimization.

Parameters
paramstuple

A tuple with the parameters of func according to their order of input

xfloat array

An independent variable.

yfloat array

The dependent variable.

funcfunction

A function with inputs: (x, *params)

Returns
float array

The marginals of the fit to x/y given the params
